# Termos de Uso - LOOMPER (Rascunho)

Este documento é um rascunho para fins de revisão jurídica. O texto final será produzido pelo departamento jurídico.

Resumo:
- LOOMPER oferece plataforma tecnológica para conexão entre contratantes (motoristas/transportadoras) e prestadores de serviços (chapas/ajudantes).
- LOOMPER atua apenas como intermediário tecnológico; não há vínculo empregatício entre as partes.
- Responsabilidades: qualidade das informações fornecidas pelos usuários; LOOMPER não se responsabiliza por atos de terceiros.
- Política de utilização, reprimendas e procedimentos para denúncias serão inseridos na versão final.
